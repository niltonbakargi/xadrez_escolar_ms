# Checklist LGPD / ECA / Lei Fléxa

## LGPD (Lei 13.709/2018)

- [ ] Autorização física assinada pelo responsável legal, arquivada pela escola
- [ ] Email e contato cadastrados são do responsável, não do aluno
- [ ] Coleta mínima — apenas dados necessários ao treinamento
- [ ] Política de privacidade em linguagem acessível exibida no cadastro
- [ ] Botão de exclusão de dados disponível ao responsável
- [ ] Dados armazenados em servidor no Brasil ou com cláusulas de transferência internacional
- [ ] Relatório de impacto à proteção de dados (RIPD) — obrigatório para dados de crianças

## ECA (Lei 8.069/1990)

- [ ] Uso de imagem somente com autorização específica e separada
- [ ] Nenhuma coleta de dados sensíveis sem base legal expressa
- [ ] Ambiente sem contato com usuários externos à turma

## Lei Fléxa (Lei 14.460/2022)

- [ ] Monitoramento de tempo de tela implementado
- [ ] Aviso de pausa configurado por faixa etária:
  - 6–12 anos: pausa a cada 45 minutos
  - 13–17 anos: pausa a cada 60 minutos
- [ ] Chat desligado na versão inicial
- [ ] Todas as interações registradas e auditáveis
- [ ] Nenhum conteúdo de terceiros sem curadoria do professor

## Segurança de dados

- [ ] Senhas armazenadas com bcrypt (custo mínimo 12)
- [ ] Comunicação apenas via HTTPS em produção
- [ ] Tokens JWT com expiração de 8 horas
- [ ] Logs de acesso retidos por 6 meses
- [ ] Plano de resposta a incidentes documentado
